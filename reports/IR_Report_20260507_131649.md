### Loại bỏ (Eradication)

#### 🟠 P2-HIGH – Loại bỏ toàn bộ thành phần mã độc

**Danh mục:** Dọn dẹp hệ thống

Xóa 3 file đã drop, dọn dẹp registry và scheduled tasks. Chạy antivirus quét toàn bộ sau khi xử lý thủ công.

```powershell
# Quét toàn bộ hệ thống với Windows Defender (đã cập nhật):
Start-MpScan -ScanType FullScan

# Kiểm tra Scheduled Tasks độc hại:
Get-ScheduledTask | Where-Object {$_.State -ne 'Disabled'} | Format-List

# Kiểm tra dịch vụ độc hại:
Get-Service | Where-Object {$_.Status -eq 'Running'} | Select-Object Name, DisplayName

# Xóa các file mã độc đã drop:
Remove-Item -Force "C:\Users\Admin\AppData\Local\Temp\payload.dll" -ErrorAction SilentlyContinue
Remove-Item -Force "C:\ProgramData\WindowsUpdate\svchost32.exe" -ErrorAction SilentlyContinue
Remove-Item -Force "C:\Users\Admin\AppData\Roaming\Microsoft\Windows\Start Menu\Programs\Startup\updater.lnk" -ErrorAction SilentlyContinue
```

> 💡 Nếu có file được bảo vệ, restart vào Safe Mode để xóa.
> 💡 Kiểm tra Recycle Bin và %TEMP% folder.

#### 🟠 P2-HIGH – [Persistence] Kiểm tra autostart locations

**Danh mục:** MITRE ATT&CK - T1547

Phát hiện persistence thông qua autostart. Kiểm tra toàn bộ autorun locations.

```powershell
# Dùng Autoruns (Sysinternals):
autorunsc.exe -a * -o autoruns_output.csv

# PowerShell - Kiểm tra Run keys:

Get-ItemProperty HKCU:\Software\Microsoft\Windows\CurrentVersion\Run

# Scheduled Tasks:

```
